# ✨ Perfect Chaos

A generative 2D collision simulation where every ball calculates a unique path to hit moving platforms.

<p align="center">
  <a href="https://perfect-chaos-simulation.vercel.app/"><strong>Try Perfect Chaos</strong></a>
</p>

<p align="center">
  <em>One bounce can create an entirely new trajectory.</em>
</p>

## Features

- Moving platforms with predicted interception points
- Straight-line trajectories with ceiling and floor bounces
- New balls created from platform collisions
- Staggered collision timing and a 6,000-ball limit
- Responsive controls for pause, reset, and simulation speed

## Built with

- HTML5 Canvas
- Vanilla JavaScript
- CSS

---

Made as a study in motion, prediction, and controlled chaos.
